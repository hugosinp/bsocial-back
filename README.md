## Configuration
Don't forget to setup a .env file with the following keys

- DATABASE_URL
- JWT_SECRET
- JWT_EXPIRATION

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
